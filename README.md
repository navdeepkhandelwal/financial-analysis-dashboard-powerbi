# Financial Analysis Dashboard (Power BI)

## Project Overview
This project presents a Financial Analysis Dashboard built using Power BI to analyze sales performance, cost of goods sold (COGS), and profit trends across different products and segments.

The dashboard enables business stakeholders to monitor financial performance, identify profitable products, and understand revenue distribution across customer segments.

---

## Key Features
- Sales, COGS, and Profit KPI tracking
- Profit distribution across market segments
- Product-wise profit contribution
- Monthly financial trend analysis
- Interactive country and year filters

---

## Tools Used
Power BI  
DAX  
Excel  
Data Modeling  

---

## Key Insights
- Government segment contributes the highest profit share
- Certain products generate significantly higher profit margins
- Sales and COGS trends show seasonal fluctuations across months
- Profitability varies significantly across product categories
